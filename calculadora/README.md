# Calculadora 
---

- Feita com HTML, CSS e JAVASCRIPT
- O botão AC, se clicado 2x aciona o som
- Calculadora de 8 dígitos
- Projeto com o auxílio da [HCode](https://www.hcode.com.br/)
- Mostra a hora e data atual
